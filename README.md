1. Clone the repository:
```bash

```

```bash

```

```bash

```

4. Open your browser to `http://localhost:5173`

### First Time Setup

1. **Create Puppy Profile**: Enter your bernedoodle's basic information
   - Name, breed, birth date, arrival date
   - Gender, color/markings, expected adult weight

2. **Safety Setup**: Complete the cat integration safety checklist
   - Separate safe spaces
   - Baby gates and barriers
   - Elevated cat perches
   - Feeding area separation

3. **Start Logging**: Use the quick-add button to log daily activities
   - Potty breaks and training sessions
   - Meals and exercise
   - Cat interactions and milestones
