# Puppy Profile Picture

To add your puppy's profile picture:

1. Save the puppy image as `puppy-profile.jpg`
2. Place it in this folder: `C:\projects\jamie\StartProject\public\images\`
3. The image will automatically appear in the navigation

The image should be:
- Format: JPG or PNG
- Recommended size: 500x500px or larger (it will be displayed as 128x128px circular)
- File name: `puppy-profile.jpg`
