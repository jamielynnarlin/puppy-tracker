üöÄ Get Your Puppy Tracker Running

Install Node.js

Download and install Node.js from nodejs.org (choose the LTS version)

https://nodejs.org/

Open Terminal in VS Code

Press Ctrl + ` to open the integrated terminal in VS Code

Install Dependencies

Run this command to install all required packages:

npm install

Start the Development Server

Launch your puppy tracker webapp:

npm run dev

This will open your browser to http://localhost:5173 where you can start tracking your bernedoodle's journey!
